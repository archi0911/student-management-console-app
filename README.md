# Student Management Console Application

## Description

This is a simple Student Management System built using JavaScript.  
It runs in the console and performs basic CRUD operations.

CRUD means:

- Create (Add Student)
- Read (List Students)
- Update (Edit Student)
- Delete (Remove Student)

---

## Features

- Add a student
- View all students
- Search student by name
- Update student by ID
- Remove student by ID

---

## Technologies Used

- JavaScript
- Console (Browser Console)

---

## Note

- Data is stored in an array.
- Data will be lost when the program stops.
- This is a console-based project for learning purposes.
